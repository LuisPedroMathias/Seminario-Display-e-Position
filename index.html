Flow-root

Essa função “limpa” problemas de flutuação (floats). Usando à um elemento próximo a um elemento flutuante, o elemento flutuante não fica sobreposto ao outro.

Table

Similar ao Grid, a table também cria um layout em linhas e colunas, porém, por ser uma função mais antiga, permite somente dados que já sejam formatados como tabelas.

block flex

Junta as propriedades do display: block e display:flex, fazendo com que o elemento possa ficar na mesma linha de outros elementos e sempre quebre a linha. Como resultado, o elemento ocupará toda a linha e organizará os filhos como flexbox.

block flow-root

No display: block flow, com um fluxo comum, pode ocorrer vazamento de float pela altura do container não ser tão precisa. No display: block flow-root o fluxo é isolado, a altura do container é a correta e nada vaza.
