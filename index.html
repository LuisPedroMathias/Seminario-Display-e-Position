Propriedades Position e Display no CSS

Display

Display

A propriedade Display, no CSS, faz a definição do tratamento do elemento: bloco ou elemento inline e o layout usado para seus filhos (herança do CSS, onde propriedades de estilo passam de pai para filho), como layout de fluxo, grid ou flex. De maneira prática, o Display define os tipos de exibição internos e externos de um elemento.

Block

Esta propriedade fará com que seja gerada uma caixa de elemento de bloco, gerando quebras de linha automáticas antes e depois do bloco (em fluxo normal). Basicamente, os elementos ocupam toda a largura disponível e iniciam uma linha nova.

Inline

Ao contrário do block, o inline gera uma ou mais caixas de elemento em linha, sem nenhum tipo de quebra de linha. Sempre que houver espaço, o próximo elemento estará na mesma linha que os outros ( em fluxo normal). Em resumo, elementos ocupam a largura necessária e não iniciam em uma nova linha

Inline-block

Gera uma caixa de elemento de bloco que fluirá com o conteúdo ao redor, como se fosse uma caixa única em linha. Tal valor é equivalente ao valor “inline flow-root. Conclui-se que assim os elementos ocupam a largura necessária enquanto é permitido que se manipule largura e altura: ou seja, uma junção de inline e block.

Flow-root

Essa função “limpa” problemas de flutuação (floats). Usando à um elemento próximo a um elemento flutuante, o elemento flutuante não fica sobreposto ao outro.

Position

Position: Static

O elemento escolhido não é afetado pelas propriedades top, right, bottom, left, entre outras. Seguindo o fluxo normal do documento.

Teste1

Teste2

Teste3

Position: Relative

O elemento escolhido permanece no fluxo normal, porém pode ser deslocado em relação a sua própria posição original, usando top, left, right, etc. Ainda assim tendo o seu espaço original preservado.

Teste1

Teste2

Position: Absolute

O elemento escolhido tem o seu fluxo normal removido, ele se posiciona em relação ao antecessor mais próximo com position diferente de static. Caso não exista o antecessor, será necessário o uso do HTML.

Position: Fixed

O elemento escolhido é removido do fluxo normal, logo o elemento se posiciona em relação ao viewport, e não se move ao scrollar.

Position: Sticky

O elemento escolhido se comporta como relative até atingir o limite do scroll, então ele muda de estado e fica fixo, como o fixed. Sendo uma propriedade híbrida entre o relative e o fixed.
